#  AI Business Assistant

An AI-powered platform that helps startups analyze finances, simulate business scenarios, and make smart decisions.

## Features
- Budget Analysis
- Business Dashboard
- AI Decision Engine
- Multi-role AI Assistant
- Voice Input
- AI Insights
- Revenue Prediction
- PDF Report Generation

## Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)
- Machine Learning (Scikit-learn)
- Groq API (AI)

## Run Locally

### Backend
uvicorn backend:app --reload

### Frontend
streamlit run app.py
